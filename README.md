# Immersphere Pro · CRM Comercial Local

App standalone para gestión de 50 leads comerciales de **Immersphere Pro** y **Rubik SOTA**.  
Zona: Torrevieja / Orihuela Costa · 2026.

🔗 **[Abrir App](https://Juanmaes83.github.io/immersphere-pro-crm-leads/)**

---

## Funcionalidades

- **Dashboard comercial** — KPIs en tiempo real: total leads, prioridad A, score medio, pipeline estimado, estados
- **Lista de 50 leads** — Tarjetas con score, prioridad, sector, zona, estado y dolor principal
- **Filtros** — Por prioridad (A/B/C), estado comercial y búsqueda libre por empresa/sector/zona
- **Top 10 de ataque** — Ranking visual por score con gancho personalizado y servicio recomendado
- **Pipeline Kanban** — 5 columnas: Sin contactar → Contactado → Propuesta → Cerrado → No interesa
- **Ficha completa de lead** — Toda la inteligencia comercial en un modal
- **Mensajes copy-paste** — WhatsApp inicial, asunto email, cuerpo email, follow-up 1 y follow-up 2
- **Objeciones y cierres** — Respuesta preparada a la objeción probable + técnica de cierre
- **Persistencia local** — Los estados se guardan en `localStorage` y sobreviven recargas

## Stack

- HTML + CSS + JavaScript vanilla
- Sin backend
- Sin base de datos
- Sin dependencias externas
- Datos hardcodeados en JSON (50 leads)

## Despliegue

GitHub Pages — rama `main`, carpeta raíz.

## Uso

Abre `index.html` en cualquier navegador. No requiere servidor.
